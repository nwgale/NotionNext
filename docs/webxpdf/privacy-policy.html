WEBxPDF Privacy Policy

Last updated: July 17, 2026

This Privacy Policy explains how WEBxPDF handles information when users convert webpages into PDF files through the Chrome extension.

Information Processed by the Extension

WEBxPDF may access the active webpage when the user starts a PDF conversion. This access is used to read page structure, prepare content, generate the PDF, name the PDF file, and save the result through Chrome.

Data We Do Not Sell

We do not sell user data. We do not use user webpage content for unrelated advertising, profiling, or resale.

Local Settings

The extension may store user preferences locally, such as filename settings, conversion options, trial status, or other configuration needed to provide the extension experience.

Remote Services

WEBxPDF may communicate with backend services for trial management, payment verification, account status, or related service functions. Only the minimum information needed for those functions should be processed.

Page Content

Webpage content is accessed only when the user initiates conversion. The extension needs temporary access to the current page to produce the requested PDF output.

Permissions

WEBxPDF uses Chrome permissions such as activeTab, scripting, downloads, storage, contextMenus, debugger, and host permissions to provide conversion, saving, settings, and page preparation features.

Contact

For questions about privacy, please contact the developer through the Chrome Web Store listing or related product support channels.
